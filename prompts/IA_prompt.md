# Prompt Exact Soumis à l'IA

Voici le prompt exact (ou une version légèrement reformulée pour être générique) qui a été soumis au modèle d'Intelligence Artificielle pour obtenir le code HTML et CSS de la page sur les jeux vidéo :

---

**Rôle :** Agis en tant qu'ingénieur web front-end avec une passion pour le gaming et la création de documentation.

**Objectif :** Générer deux fichiers : `index.html` et `style.css`. La page web doit être informative sur l'évolution du jeu vidéo et utiliser une esthétique "gaming" (couleurs sombres, accents vifs).

**Instructions pour index.html :**
1.  Structure : Utiliser `header`, `main`, `section`, `ul` et `footer`.
2.  Contenu : Inclure un titre principal, un paragraphe sur l'évolution, une liste de 3 technologies clés (Moteurs de jeu, Versionnement, Plateformes).
3.  Image : Utiliser cette URL d'image libre de droit dans une balise `<img>` avec la classe `hero-image` : `https://images.unsplash.com/photo-1542751371-adc32f867cd0`

**Instructions pour style.css :**
1.  Palette : Utiliser un `background-color` très sombre (ex: #1a1a2e) et des couleurs d'accentuation vives (ex: #e94560 ou #4CAF50).
2.  Typographie : `font-family` sans serif.
3.  Design : Les sections doivent être séparées et avoir une légère ombre portée pour un effet 3D ou "pop-out".
4.  L'image `.hero-image` doit prendre toute la largeur de son conteneur et avoir des bords arrondis.

**Format de sortie :** Fournis d'abord le code HTML, puis le code CSS.
